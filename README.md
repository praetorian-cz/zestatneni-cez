# Kalkulačka zestátnění ČEZ

Upozornění

Tato kalkulačka je experimentální nástroj vytvořený pro ilustrativní a vzdělávací účely. Nejde o investiční doporučení, analýzu, ani nabídku k nákupu či prodeji jakýchkoli cenných papírů.

Všechny výpočty vycházejí z veřejně dostupných údajů a zjednodušujících předpokladů, které nemusí odpovídat skutečnosti. Vstupní parametry si volí uživatel sám a zobrazené hodnoty jsou pouze modelové — nevyjadřují skutečnou, schválenou ani očekávanou výkupní cenu a nemusí znamenat vůbec nic.

Autor neručí za správnost, úplnost ani aktuálnost údajů a nenese žádnou odpovědnost za jakákoli rozhodnutí učiněná na základě tohoto nástroje. Použití je výhradně na vlastní riziko.
